-----------------------------------------------

3. Send single question request from dedup:
-------------------------------------------
curl -X GET http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"uri1":"http://vocab.getty.edu/ulan/500028092","uri2":"http://edan.si.edu/saam/id/person-institution/1681","dedup":{"ver":"1.0","revision":"4","score":"0.45"}}'

4. Send multiple question request from dedup:
---------------------------------------------
curl -X GET http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"bulk":10,"payload":"[{"uri1":"http://vocab.getty.edu/ulan/500028092","uri2":"http://edan.si.edu/saam/id/person-institution/1681"},"dedup":"{}",...10 enties...]"}'

5. Send submit answer request 
-----------------------------------------------
curl -X PUT http://localhost:5000/v1/answer -H "Content-Type: application/json" -d '{"value":"1","comment":"Just another comment","qid":"56fdecf6f6bf2d187445d16b"}'
