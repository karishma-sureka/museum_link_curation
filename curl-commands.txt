Curl Commands
=============

1. Send single question request from interface (GET request):
-------------------------------------------------------------
curl -X GET http://localhost:5000/v1/question -u nilayvac@usc.edu:linkCuration
curl -X GET http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"stats":"True"}' -u nilayvac@usc.edu:linkCuration

2. Send bulk question request from interface (GET request):
-----------------------------------------------------------
curl -X GET http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"bulk":10}' -u nilayvac@usc.edu:linkCuration
curl -X GET http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"bulk":10,"stats":"True"}' -u nilayvac@usc.edu:linkCuration

3. Send single question request from dedupe (POST request):
----------------------------------------------------------
curl -X POST http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"uri1":"http://vocab.getty.edu/ulan/50001234","uri2":"http://edan.si.edu/saam/id/person-institution/1234","dedupe":{"ver":"1.0","revision":"4","score":"0.45"}}'

4. Send multiple question request from dedupe (POST request):
-------------------------------------------------------------
-- Format -- 
curl -X POST http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"bulk":<n>,"payload":[{},{},... n entries]}'

-- Example -- 
curl -X POST http://localhost:5000/v1/question -H "Content-Type: application/json" -d '{"bulk":2,"payload":[{"uri1":"http://vocab.getty.edu/ulan/50002345","uri2":"http://edan.si.edu/saam/id/person-institution/2345","dedupe":{"ver":"1.0","revision":"4","score":"0.45"}},{"uri1":"http://vocab.getty.edu/ulan/50003456","uri2":"http://edan.si.edu/saam/id/person-institution/3456","dedupe":{"ver":"1.0","revision":"4","score":"0.85"}}]}'

6. Register User (POST request)
-------------------------------
curl -X POST http://localhost:5000/v1/user -H "Content-Type: application/json" -d '{"username":"nilayvac@usc.edu","password":"linkCuration"}'
curl -X POST http://localhost:5000/v1/user -H "Content-Type: application/json" -d '{"username":"ksureka@usc.edu","password":"linkCuration"}'

7. Login User (for duration) and get time bound API token (GET request)
------------------------------------------------------------------
curl -u nilayvac@usc.edu:linkCuration -X GET http://localhost:5000/v1/user -H "Content-Type: application/json"
curl -u nilayvac@usc.edu:linkCuration -X GET http://localhost:5000/v1/user -H "Content-Type: application/json" -d '{"duration":3600}'
